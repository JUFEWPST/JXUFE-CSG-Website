# 江西财经大学网络安全协会网站建设项目

## 概述

本项目用于开发协会的网站，采用基于 [Nuxt.js](<(https://nuxt.com.cn)>) 的现代化技术栈

目前网站计划建设如下板块

- 首页（狠狠给协会打 cool，然后简略介绍协会现状）
  举例：从上往下分别是 大大的网安协会字样、近期动态、协会简介
- 具体介绍页
  包含协会荣誉、历届成员、成员荣誉等
- 博客页
  用于发布协会的新闻/技术博客
- 加入我们
  贴个二维码也不是不行
- 某个不知名角落的致谢

如有更多建议请提交 Issue 或者直接来 PR

目前该项目已经搭了个实例页面，有请大家 PR 吧
